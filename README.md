# Posts Django Project
Це Django-проєкт для роботи з постами.
---

## Технології

* Python 3.13
* Django 6.0.3

---

## Як запустити проєкт

1. Клонувати репозиторій:
   git clone (https://github.com/ohnista-lks07/djangoPost)

2. Перейти в папку проєкту:
   cd cinema_django

3. Створити віртуальне середовище:
   python -m venv .venv

4. Активувати середовище:
   Mac:
   source .venv/bin/activate

5. Встановити Django:
   pip install django

6. Застосувати міграції:
   python manage.py migrate

7. Запустити сервер:
   python manage.py runserver
